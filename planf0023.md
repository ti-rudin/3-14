# План улучшения лендинга АО "Полезные ископаемые"

## 1. Производительность и оптимизация изображений

### Оптимизация изображений

- Добавить автоматическую конвертацию изображений в WebP/AVIF форматы
- Внедрить `<picture>` элементы с fallback для старых браузеров
- Реализовать responsive images с srcset для разных разрешений
- Добавить placeholder/blur эффект во время загрузки (skeleton screen)
